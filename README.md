<h4 align="center">Convenient and simple image generation with Fusion Brain API</h4>

## Available languages 🌍
- Russian 🇷🇺
- English 🇬🇧

## Features 📱
- Сhoosing different styles from the ready-made set ✅
- Saving favourites to the special app album ❤️
- Saving images to the device's gallery 💾
- Theme: Dark | Light 🖤🤍

## Technologies stack 🚀
- Single Activity
- MVI Architecture
- Clean Architecture
- Jetpack Compose & Material Design 3
- Kotlin Coroutines & Flow
- Compose Navigation
- Retrofit2 + OkHttp
- Room Database
- Dagger Hilt
- Android test
